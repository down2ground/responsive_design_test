About this page

This page demonstrates different Markdown features.

Footnotes

Python-Markdown 3.4.4 documentation: Footnotes.

Footnote links are added by the following expression [^1] as in the example[^1].

The footnotes themselves are added by the expressions like [^1]: The content of the footnote 1.. They are added to the place marked by ///Footnotes Go Here/// text or in the page end.

[^1]: The content of the footnote 1.

///Footnotes Go Here///

The second marker does not work. Here's the second link[^2]. And the footnote content will be above, not below.

[^2]: The content of the footnote 2.

///Footnotes Go Here///

For now it works only in the Python implementation.
